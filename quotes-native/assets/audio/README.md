# Audio Assets

This directory contains audio notification files for quote transitions.

## Required File

- `notification.mp3` - Buddhist bell sound or gentle notification (<100KB)

## Sources

You can use:
1. A Buddhist temple bell sound
2. Singing bowl sound
3. Gentle chime

Recommended: Find a royalty-free Buddhist bell sound from sources like:
- freesound.org
- zapsplat.com
- soundbible.com

The file should be:
- Format: MP3
- Duration: 0.5-2 seconds
- Size: <100KB
- Volume: Moderate (will be user-adjustable)

## Note for Development

For now, create a placeholder or download a suitable sound. The app will gracefully handle missing audio files in development mode.
